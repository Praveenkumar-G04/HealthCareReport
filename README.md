# Power BI Dashboard Project – Hospital Patient Waitlist Analysis

Excited to share a Power BI dashboard I built to analyze hospital patient waitlist data and provide meaningful insights for healthcare decision-making.

This dashboard helps visualize:

Key Insights
• Latest Month Waitlist vs Previous Year comparison
• Patient distribution by Case Type (Outpatient, Day Case, Inpatient)
• Waitlist trends across time bands (0–3 months, 3–6 months, etc.)
• Age profile analysis of patients
• Specialty-wise patient waitlist insights
• Monthly trend analysis for patient volume

Tools & Technologies Used
• Power BI
• Data Modeling
• DAX
• Power Query
• Interactive Dashboard Design

This project demonstrates how data visualization can support healthcare planning and resource optimization.
